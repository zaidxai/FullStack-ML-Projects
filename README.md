# 🚀 FullStack-ML-Projects

A collection of **end-to-end Machine Learning projects** combining model training, backend APIs, and frontend applications.

This repository focuses on building **real-world deployable ML systems** rather than isolated notebooks.

---

## 🧠 What This Repository Demonstrates

* End-to-end ML pipeline development
* Integration of ML models with FastAPI backend
* Frontend + ML system interaction
* Real-time inference systems
* Deployment using Hugging Face

---

## 📌 Featured Projects

### 📧 Spam Mail Detection System

A BERT-based NLP classifier that detects spam messages with real-time predictions using a FastAPI backend and React frontend.

---

### 🌡️ Temperature Prediction System

A regression-based ML system for predicting temperature trends using historical data.

---

### 🏠 House Price Prediction System

## 🚀 How to Run Any Project

Each project in this repository is independent and contains its own setup and dependencies.

---

### 🔹 Backend Setup (FastAPI)

bash
cd spam-detection/backend

# create virtual environment
python -m venv venv

# activate virtual environment (Windows)
venv\Scripts\activate

# install dependencies
pip install -r requirements.txt

# run FastAPI server
uvicorn main:app --reload

---

### 🔹 Frontend Setup (React)

bash
cd spam-detection/frontend

# install dependencies
npm install

# start development server
npm start
